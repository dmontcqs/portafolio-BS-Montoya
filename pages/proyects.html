MVE webBook

MVE webBook es la versión web de un ensayo académico universitario. Mi interés en este proyecto donde utilicé HTML y CSS es explorar las posibilidades del diseño y la tipografía web. Esto ayuda a que el contenido sea más accesible a públicos amplios. Realizar un layout responsive también fue uno de mis cometidos principales.

Twombly´s monography

En este proyecto utilicé el ensayo de la académica Sally Nexen en su trabajo sobre Cy twombly "Land of Stars" para explorar las obras del pintor estadounidense dentro de una página web. Con CSS construí el diseño responsive que también tiene interacción con el usuario pues despliega información al posicionarse sobre la obra. También agregué un botón de "modo oscuro" para hacer el sitio más accesible.

Lasker variations

En este proyecto interactivo utilicé la composición visual de la pintura "Art and Meaning" (2015) del artista estadounidense Jonathan Lasker para que el usuario realice diferentes versiones de la pintura y pueda guardar su propia composición. En este proyecto me apoyé de tecnologías como HTML, CSS, Javascript lo mismo que jQuery que ayudó a refactorizar el código.

Russian School

Russian School es un proyecto para explorar la interacción de algoritmos de Javascript. Para su funcionalidad, utilicé jQuery e implementé algunas animaciones con esa misma libería. También integré una API de clima que muestra en tiempo real el clima de la ciudad de Moscú. Parte del layout está realizado con Bootstrap.
